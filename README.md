# MNIST Digit Classification

Proyecto académico realizado para la materia Inteligencia Artificial (2024).

El objetivo del trabajo es entrenar y evaluar modelos de aprendizaje supervisado
para la clasificación de dígitos escritos a mano utilizando el dataset MNIST.

## Contenidos
- Exploración y visualización del dataset
- Preprocesamiento de imágenes
- Entrenamiento de modelos de clasificación
- Evaluación de desempeño

## Tecnologías utilizadas
- Python
- Google Colab
- NumPy
- Matplotlib
- Scikit-learn
